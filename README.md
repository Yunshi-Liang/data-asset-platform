# 电力设计院数据资产管理平台 Demo

## 项目简介

本项目是面向电力设计院数据资产管理场景的前端演示系统，展示数据从接入、治理、数据资产目录与资产详情、数据产品上架，到数据产品门户申请和个人工作台审批的完整业务流程。

## 技术栈

- React（JavaScript）
- Vite
- Ant Design
- React Router
- 前端 Mock 数据

## 当前功能

- 数据资产概览首页
- 数据产品门户与产品检索申请
- 多源数据接入与任务执行模拟
- 数据治理流程与质量问题处理
- 数据资产目录与完整资产画像
- 数据产品审核及上架发布
- 个人工作台、数据申请与审批

## 本地运行

```bash
npm install
npm run dev
npm run build
```

## 项目性质

本项目用于面试现场演示，重点呈现产品设计、业务理解和前端交互能力，不包含真实后端服务，也不作为生产系统使用。

## 数据说明

项目中的业务系统、工程项目、用户及运营数据均为模拟数据，不代表任何真实企业、人员或项目。
